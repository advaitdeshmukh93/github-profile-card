### Executable Scripts Reference

```bash
# Development server initialisation with hot module replacement
pnpm dev

# TypeScript compilation sans code emission
pnpm build

# Test suite execution
pnpm test

# Test suite execution in continuous observation mode
pnpm test:watch

# Code coverage analysis report generation
pnpm test:coverage

# Code quality assessment
pnpm lint

# Automated linting deficiency rectification
pnpm lint:fix

# Code formatting standardisation
pnpm format

# Code formatting standardisation verification
pnpm format:check

# Type annotation validation
pnpm typecheck
```
